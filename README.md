# 산책어플 프로젝트

# 1. 프로젝트 개요
- 프로젝트 이름: 산책 경로 공유 어플
- 프로젝트 설명: 새로운 산책길을 찾거나 나의 산책길을 공유하는 공간입니다.

# 3. 주요 기능

- **나만의 산책길을 기록**:
  -남들에게 알리고 싶은 나의 산책길을 기록해 게시판에 공유할 수 있다.

- **산책길  공유**:
  -게시판에서 주변 산책길이나 인기 산책길을 둘러볼 수 있다.

- **추천 산책길 안내**:
  -선택한 산책길에 대한 자세한 경로와 이미지를 확인하고 산책길 안내를 받을 수 있다.
